## 🔧 TROUBLESHOOTING

### If Backend Window Closes:
1. Double-click `start-server.bat` again
2. Wait 3 seconds
3. Refresh browser (F5)

### If Demo Page Won't Load:
1. Make sure backend is running (black window open)
2. Close and reopen `demo.html`
3. Check browser console (F12)

### If Features Don't Work:
1. Refresh the page (F5)
2. Make sure you see "Connected to server" notification
3. Look for green "Live" indicator
